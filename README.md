# 减肥搭子微信机器人 MVP

## 项目概述
- **定位**：AI减肥搭子微信机器人
- **核心价值**：12种人格陪伴 + 24小时响应 + 无社交压力
- **目标用户**：想减肥但无法坚持的I人（内向者）

## 技术架构（7模块）
1. 微信机器人框架 (wechat_bot.py)
2. AI模型集成-文心一言版 (ernie_integration.py)  
3. 人格管理器 (personality_manager.py)
4. 减肥分析模块 (weight_analysis.py)
5. 激励语生成器 (motivation_generator.py)
6. 云数据库集成-MongoDB Atlas (cloud_storage.py)
7. 配置管理系统 (app_config.py)

## 核心算法
- 精准记忆检索（四维检索）
- 成长性对话（目标管理+阶段语言）
- 记忆管理（三层记忆架构）
- 人格一致性保障
- 去AI化真人回应
